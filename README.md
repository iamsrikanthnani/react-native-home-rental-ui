### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iamsrikanthnani/react-native-home-rental-ui.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Add `local.properties` in `root/android`
   ```sh
   sdk.dir = /path/to/Android/Sdk
   ```
4. Run
   ```sh
   react-native run-android and npx start
   ```
